Btcbf

A fast and efficient Bitcoin private key brute force tool

View on GitHub Download Latest Release

Features

Fast & Efficient

Optimized for speed using the bit library for Bitcoin operations

Online & Offline Modes

Check addresses through an online API or an offline database

Multiple Methods

Generate random or sequential private keys and check their balances

Multi-threaded

Utilizes multiple CPU cores for improved performance

Security Analysis

Demonstrates Bitcoin's cryptographic security through practical experimentation

Interactive CLI

User-friendly command-line interface with guided prompts and instructions

Quick Start

$ git clone https://github.com/vlnahp/Btcbf.git
$ cd Btcbf
$ pip install -r requirements.txt
$ python Btcbf.py  # or python3 Btcbf.py on Linux

Usage

Just execute the command python Btcbf.py or python3 Btcbf.py on Linux. The program will guide you through the process with interactive prompts.

Results will be saved to foundkey.txt in the main directory.

Requirements

In offline mode, a database is necessary. By default, it is address.txt containing some addresses.

Users who wish can download the latest text file from addresses.loyce.club, rename and replace it with the "address.txt". Be careful about memory issues - only use this database if you have sufficient RAM!

About the Project

The main goal of Btcbf is to learn Python while exploring Bitcoin security concepts. The tool demonstrates the practical impossibility of brute-forcing Bitcoin private keys, highlighting the security of the Bitcoin network.
